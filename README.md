# 📱 Projeto Android - Desafio 10 (Curso em Vídeo)

Este projeto foi desenvolvido como parte do Desafio 10 do curso de **HTML5 e CSS3** do professor Gustavo Guanabara (Canal Curso em Vídeo). O objetivo foi recriar um site informativo sobre a história do mascote do Android, aplicando conceitos avançados de design web.

## 🎯 Foco do Desenvolvimento

Neste desafio, dediquei meus estudos para ir além da estrutura básica, focando em dois pilares fundamentais do Front-end:

### 1. Responsividade (Adaptação de Telas)
O site foi construído para ser totalmente flexível. Apliquei técnicas de:
* **Media Queries:** Para garantir que o conteúdo seja legível e bem diagramado tanto em monitores grandes quanto em dispositivos móveis.
* **Unidades Relativas:** Uso de `vw`, `vh`, `%` e `em/rem` para que os elementos se ajustem proporcionalmente ao tamanho da tela.

### 2. Controle de Modelos de Caixas (Box Model)
Trabalhei intensamente no comportamento dos elementos na página, dominando:
* **Margens e Preenchimentos (Margin/Padding):** Para criar um respiro visual e hierarquia entre os conteúdos.
* **Larguras Máximas (Max-width):** Utilizadas para garantir que o site não "quebre" em telas muito largas, mantendo uma leitura confortável.
* **Border-radius e Sombras:** Aplicação de detalhes estéticos que exigem precisão no cálculo do espaço das caixas.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica de todo o conteúdo.
* **CSS3:** Estilização avançada, uso de variáveis para cores e fontes personalizadas.
* **Metodologia Mobile-First:** Pensar primeiro na experiência em telas menores.
